# AmpsterWheel Project Components

## Components Overview

1. **Arduino Nano**
   - Microcontroller for processing and control

2. **VL53L0X Time-of-Flight Sensor**
   - Detects wheel rotations by sensing a hole in the wheel

3. **FS90R Continuous Rotation Servo**
   - Used for treat dispensing mechanism

4. **17HS4023 Nema 17 Stepper Motor**
   - Acts as a generator to power the system

5. **ELEGOO 0.96 Inch OLED Display**
   - Displays system stats and battery information

6. **WS2812B LED Strip**
   - Provides visual feedback

7. **Geekworm X-UPS1 Power Bank**
   - Manages power supply and battery backup

8. **18650 Battery**
   - Provides power storage

9. **Bridge Rectifier**
   - Converts AC from the generator to DC

10. **Transistor Switch**
    - Allows manual control of power from the generator

## Additional Information

- The system is designed to be modular and stackable, allowing for easy upgrades and maintenance.
- Each component is chosen for its compatibility and efficiency in the overall system.
